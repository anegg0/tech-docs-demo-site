## Wallet creation workflow

When a user is created in the marketplace (usually at the time of email activation), an inversor is created as well as a custodial wallet for them.

When an investor/issuer is created you can choose to have a custodial or not custodial wallet in onyze with an associated Public & private key for a Gr0wing Network account.
