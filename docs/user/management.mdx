### Change password

Changes user password

#### Request URL

<Highlight color="#079CEE">POST</Highlight> <code>/api/v1/public/on-boarding/change-password</code>


#### Request Body Fields

The fields below should be formatted as `json` elements and put in the request body as shown in the example below.

| Field                  | Required  | Description |
|:-----------------------|:----------|:------------|
| `password`             | Yes       | The password user wants to switch to |
| `token`                | Yes       | ???????  |

#### Example

<Tabs

  defaultValue="request"

### Delete user
Performing this action calls the backoffice to perform the anonymisation of the user (not a real delete). Anonymise user (on the backoffice) removes permanently all personal data (keeping non-personal information) to be GDPR compliant

#### Request URL
