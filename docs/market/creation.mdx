This guide walks you through the market creation process:

## Create a market

#### Request URL

<Highlight color="#079CEE">POST</Highlight> <code>/api/v1/markets</code>

#### Request Body Fields

The fields below should be formatted as `json` elements and put in the request body as shown in the example below.

| Field       | Required | Description                                          |
| :---------- | :------- | :--------------------------------------------------- |
| `networkId` | Yes      | The chosen network where the market will be deployed |
| `type`      | Yes      | PRIMARY or SECONDARY                                 |
| `name`      | Yes      | Arbitrary `string` assigning a name to the market    |
